# libft
This is the very first C project I ever made\
This 42's curriculum exercise implements the following C standard library functions

- atoi
- bzero
- calloc
- isalnum
- isalpha
- isascii
- isdigit
- isprint
- itoa
- memchr
- memcmp
- memcpy
- memmove
- memset
- strchr
- strdup
- strlcat
- strlcpy
- strlen
- strncmp
- strnstr
- strrchr
- toupper
- tolower

It also includes linked list and binary tree implementations
